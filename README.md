# Hotels-Governance (Data Governance)
## Dataset Link on Kaggle [Global Hotels](https://www.kaggle.com/datasets/raj713335/tbo-hotels-dataset)

The first phase focuses on Data Quality and Profiling, where we utilized Sweetviz for extensive data profiling, cleaned the initial 1 million records down to a highly reliable 651,079 rows by handling missing values and dropping high-null columns, built a robust data validation schema using Pandera, and designed an exploratory data visualizations using matplotlib and pandas to visualize key insights. The second phase emphasizes Privacy, Security, and Compliance, implementing SHA-256 hashing for sensitive phone numbers, a custom RSA encryption and decryption algorithm built entirely from scratch to secure hotel names, and data sketches (HyperLogLog and KLL) for CCPA-compliant data estimation. Furthermore, we integrated Attribute-Based and Role-Based Access Control (ABAC/RBAC) using the py_abac library alongside a comprehensive audit log to monitor unauthorized access attempts in compliance with HIPAA guidelines. The project also includes two advanced bonus features: a custom cryptographic attack script that factors our custom RSA modulus to successfully decrypt the ciphered data, and a complete data preprocessing pipeline (including label encoding and numeric mapping) to prepare the dataset for machine learning models.
